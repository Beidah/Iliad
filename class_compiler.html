Detailed Description

Converts text to byte code.

The compiler is takes the source code and hands it off to the Scanner to be tokenized. As the Scanner does this, the compiler takes the tokens and makes sure they occur in a grammatical sequence that can be understood by the VM. It then writes a bytecode representation of the source code to a Chunk provided by the VM.
