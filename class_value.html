Detailed Description

Basical value representation.

Values are stored in an std::vector of unsigned chars of the binary of the value. This class stores the bytes, size and an enum of the type of data that is represented. Basic arithmetic operators are overloaded to make using values in the compiler code as easy as using normal data-types. Values are stored in little endian.
