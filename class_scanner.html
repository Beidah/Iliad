Detailed Description

A class to scan through the source code and tokenize it.

The scanner class is initilized with a string containing the code to be tokenize. After initilization, ScanToken can be called to recieve each token in order of occurance, one at a time. When the scanner reaches the end of the file, ScanToken will continue to produce an EoF token each time it's called.
